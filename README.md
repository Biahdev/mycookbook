## 📋 Sobre o projeto

My CookBook - É uma plataforma de compartilhamento de receitas.

<!-- ## 🖥️ Demo -->

## ✨ Features
- Criar e compartilhar suas receitas
- Salvar suas receitas favoritas
- Comentar e dar notas sobre outra receitas
